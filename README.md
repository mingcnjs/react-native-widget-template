# Archie
Archie Home Task Assessment

## Quick Start

1. [Setup React Native Environment](https://reactnative.dev/docs/environment-setup)
2. Setup Node Environment

   ```bash
   yarn install
   ```
3. Run Metro

   ```bash
   yarn start
   ```
4. Run React Native

   **Running iOS**

   ```bash
   cd ios && pod install && cd ..`
   yarn ios
   ```

   **Running Android**

   ```bash
   yarn android
   ```
5. Clean development environment

    **Clean yarn**

    ```bash
    yarn yarn:clean
    ```
    **Clean ios**

    ```bash
    yarn ios:clean
    ```
    **Clean android**

    ```bash
    yarn android:clean
    ```
